# Meeting Costs

Real-time meeting cost calculator with employer on-costs.

Meetings have a cost. Every attendee has a salary, and the clock is running. This tool makes that cost visible to encourage better decisions: shorter meetings, fewer unnecessary attendees, and a bias toward asynchronous communication when a meeting isn't strictly needed. If you're a consultant, it's also a straightforward way to show clients why they don't need a weekly check-in when a short email would do.

## Features

- Simple mode: enter number of people and average salary, start timer
- Advanced mode: per-person salary input
- Employer on-costs: Generic UK rates (NI, auto-enrolment pension, apprenticeship levy)
- Dark mode (follows system preference)

## Privacy

No data leaves your browser. No accounts. No tracking. No cookies.

## Development

```bash
npm install
npm run dev
```

## Data Sources & Attribution

UK employer on-costs (National Insurance, pension, apprenticeship levy) are
calculated from published HMRC rates.

- UK employer NI: HMRC 2025/26
- Auto-enrolment pension: 3% on qualifying earnings (£6,240–£50,270)
- Apprenticeship levy: 0.5%
